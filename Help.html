Node Graph Controls

Adding Nodes

To add nodes to your project, use the the right click functionality of your mouse/mouse-pad while hovering over empty space with your cursor. 

This will open a menu for you to select your desired node.

Arranging Nodes

To move nodes, simply left click and drag. Multiple node selection is also suported by holdinh SHIFT and then left clicking desired nodes. By holding SHIFT while dragging these nodes, the selected nodes will move together.

To delete nodes use the DELETE key.

Editing Nodes

To edit a node, select it with left click. The selected node's properties and assigned tasks will be displayed and editable in the Node Editor on the left side of the app.

Node Types

Folder

This node holds other folder and task nodes. It's only editable property is it's name.

Singular Task

This node holds simple text. 

List Task

This node contains one or more Singular Tasks. Once all have been completed, it is mark as done.
